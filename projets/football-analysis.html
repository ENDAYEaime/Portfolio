Analyse des Matchs de Football Européens

Présentation du projet

Analyse approfondie des résultats des matchs de football européens. Ce projet utilise des techniques avancées d'analyse de données pour :

  • Analyser les performances : étude statistique des résultats des matchs et des tendances par équipe
  • Visualiser les données : création de graphiques et visualisations pour illustrer les tendances clés
  • Prédire les résultats : analyse prédictive basée sur les performances historiques
  • Identifier les patterns : découverte de motifs dans les données de matchs

Technologies utilisées

  • Python
  • Pandas
  • Matplotlib / Seaborn
  • Jupyter Notebook

Extrait du code


def analyser_performances_equipe(df, equipe):
    """Analyse les performances d'une équipe spécifique"""
    matchs_equipe = df[(df['home_team'] == equipe) | (df['away_team'] == equipe)]
    total_matchs = len(matchs_equipe)
    victoires = len(matchs_equipe[
        ((matchs_equipe['home_team'] == equipe) & (matchs_equipe['home_score'] > matchs_equipe['away_score'])) |
        ((matchs_equipe['away_team'] == equipe) & (matchs_equipe['away_score'] > matchs_equipe['home_score']))
    ])
    return {
        'total_matchs': total_matchs,
        'victoires': victoires,
        'ratio_victoires': victoires / total_matchs
    }
