Analyse de Données de Contacts

Présentation du projet

Développement d'un carnet d'adresses en Python dans le cadre du Module 1. Ce projet met en œuvre la programmation orientée objet pour créer une application permettant de :

  • Gérer les contacts : création, stockage et manipulation des informations personnelles (nom, prénom, date de naissance, adresse, téléphone, email)
  • Valider les données : vérification automatique des formats de date, email et numéro de téléphone avec des expressions régulières
  • Analyser les profils : calcul automatique de l'âge, détection du statut mineur/majeur, et estimation du genre basée sur le prénom
  • Générer des statistiques : affichage formaté des données avec gestion des erreurs robuste

Technologies utilisées

  • Python : langage principal
  • datetime : gestion des dates et calculs d'âge
  • re : validation par expressions régulières
  • Jupyter Notebook : environnement de développement

Extrait du code



            class Contact:
            def __init__(self, nom: str, prenom: str, d_anniv: str, adresse: str, telephone: str, email: str):
                self.nom = nom
                self.prenom = prenom
                self.d_anniv = datetime.datetime.strptime(d_anniv, '%d/%m/%Y')
                self.adresse = adresse
                self.telephone = telephone
                self.email = email
        
            def age(self) -> int:
                """Cette fonction determine l'âge du contact"""
                today = datetime.now()
                return today.year - self.d_anniv.year - ((today.month, today.day) < (self.d_anniv.month, self.d_anniv.day))
        
            def is_mineur(self) -> bool:
                """Cette fonction determine la majorité en fonction de l'age"""
                return self.age() < 18
        
            def is_homme(self) -> bool:
                """Cette fonction determine le sexe"""
                return self.prenom[-1].lower() != 'a'
