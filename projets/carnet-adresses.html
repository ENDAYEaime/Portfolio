Carnet d'Adresses en Python

À propos du projet

Ce projet est une application de gestion de contacts développée en Python. Elle permet de :

  • Ajouter des contacts avec leurs informations complètes
  • Rechercher des contacts par nom
  • Supprimer des contacts
  • Afficher tous les contacts du carnet
  • Validation automatique des données (email, numéro de téléphone, date)

L'application utilise la programmation orientée objet avec deux classes principales :

  • Contact : Pour gérer les informations individuelles des contacts
  • Carnet : Pour gérer la collection de contacts et les opérations CRUD

Technologies utilisées

  • Python
  • Programmation OOP
  • Module datetime
  • Expressions régulières (re)
  • Jupyter Notebook

Fonctionnalités clés

  • Interface en ligne de commande interactive
  • Validation des entrées utilisateur
  • Détection automatique de l'âge et du statut (mineur/majeur)
  • Gestion des erreurs robuste
  • Structure de données optimisée

class Contact:
    def __init__(self, nom: str, prenom: str, d_anniv: str, 
                 adresse: str, telephone: str, email: str):
        self.nom = nom
        self.prenom = prenom
        self.d_anniv = datetime.datetime.strptime(d_anniv, '%d/%m/%Y')
        self.adresse = adresse
        self.telephone = telephone
        self.email = email

    def age(self) -> int:
        today = datetime.now()
        return today.year - self.d_anniv.year - (
            (today.month, today.day) < 
            (self.d_anniv.month, self.d_anniv.day)
        )

    def is_mineur(self) -> bool:
        return self.age() < 18
