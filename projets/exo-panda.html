Exercices Pandas

Présentation du projet

Ce projet présente une série d'exercices pour apprendre et pratiquer l'utilisation de la bibliothèque Pandas en Python. Les exercices couvrent divers aspects de la manipulation de données et de l'analyse statistique.

  • Manipulation de données : chargement, nettoyage et transformation des données
  • Analyse statistique : calculs de statistiques descriptives et exploration des données
  • Visualisation des données : génération de graphiques pour illustrer les résultats

Technologies utilisées

  • Python : langage principal
  • Pandas : bibliothèque pour la manipulation de données
  • Matplotlib / Seaborn : bibliothèques pour la visualisation
  • Jupyter Notebook : environnement de développement
