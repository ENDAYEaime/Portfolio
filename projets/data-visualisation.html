Visualisation des Centrales Électriques Européennes

Présentation du projet

Projet d'analyse et de visualisation de données sur les centrales électriques européennes. Ce projet utilise plusieurs bibliothèques de visualisation Python pour créer des représentations interactives et informatives des données énergétiques à travers l'Europe.

  • Analyse complète : Exploration des données sur les centrales électriques
  • Visualisations multiples : Utilisation de Matplotlib, Seaborn et Bokeh
  • Interactivité : Création de graphiques interactifs
  • Insights : Extraction de tendances et patterns

Technologies utilisées

  • Python : Langage principal
  • Pandas : Manipulation des données
  • Matplotlib : Visualisations de base
  • Seaborn : Visualisations statistiques
  • Bokeh : Visualisations interactives

Extrait du code


            import pandas as pd
            import matplotlib.pyplot as plt
            import seaborn as sns
            import bokeh

            # Lecture des données
            df = pd.read_csv("powerplants-eu.csv")

            # Configuration du style
            plt.style.use('seaborn')
            sns.set_palette("husl")

            # Création de visualisations
            def create_capacity_by_source():
                plt.figure(figsize=(12, 6))
                sns.barplot(
                    data=df,
                    x='energy_source',
                    y='capacity',
                    estimator=sum
                )
                plt.xticks(rotation=45)
                plt.title('Capacité totale par source d\'énergie')
                plt.tight_layout()
                return plt
