Dashboard Évolutif - Capacité Crânienne Humaine

Présentation du projet

Application Streamlit interactive et modulaire pour explorer l'évolution de la capacité crânienne humaine à travers le temps, l'espace et les facteurs morphologiques. Le dashboard propose :

  • Analyse statistique : Statistiques descriptives, catégorielles et exploration du dataset
  • Visualisations avancées : Histogrammes, boxplots, corrélations, évolution temporelle, analyse multivariée
  • Carte interactive : Répartition mondiale des observations par pays (Folium)
  • Synthèse & recommandations : Résultats clés, conclusions, axes de recherche futurs

Technologies utilisées

  • Streamlit : Interface web interactive
  • Python : Traitement et analyse
  • Pandas : Manipulation de données
  • Plotly, Seaborn, Matplotlib : Visualisations
  • Folium : Cartographie interactive

Extrait du code


import streamlit as st
import pandas as pd
import plotly.express as px

st.set_page_config(page_title="Dashboard - Accueil", layout="wide")

df = pd.read_csv('Evolution_DataSets_clean.csv')
st.title("Évolution de la Capacité Crânienne chez l'Homme")

# Statistiques clés
col1, col2, col3 = st.columns(3)
col1.metric("Spécimens", f"{len(df):,}")
col2.metric("Pays analysés", df['Current_Country'].nunique())
col3.metric("Capacité moyenne", f"{df['Cranial_Capacity_(cm³)'].mean():.0f} cm³")

# Visualisation temporelle
fig = px.scatter(df, x='Time_MYA', y='Cranial_Capacity_(cm³)',
                 title="Évolution de la Capacité Crânienne au Fil du Temps",
                 labels={'Time_MYA': "Temps (Millions d'années)", 'Cranial_Capacity_(cm³)': "Capacité Crânienne (cm³)"},
                 trendline="ols")
fig.update_layout(xaxis=dict(autorange='reversed'))
st.plotly_chart(fig, use_container_width=True)
