Exploration des Publications Hacker News

Présentation du projet

Ce projet compare deux types de publications sur Hacker News : `Ask HN` et `Show HN`. L'objectif est de déterminer lequel de ces types reçoit le plus de commentaires en moyenne et d'identifier les heures optimales pour publier afin de maximiser les commentaires.

  • Analyse des commentaires : comparaison des commentaires moyens entre `Ask HN` et `Show HN`
  • Optimisation du temps de publication : identification des heures de publication optimales
  • Analyse statistique : Étude de la donnée pour analyser et représenter les insights obtenus

Technologies utilisées

  • Python : langage principal
  • Pandas : manipulation et analyse de données
  • Jupyter Notebook : environnement de développement

Extrait du code



            import csv

            # Lecture des données
            with open('hacker_news.csv') as f:
                hn = list(csv.reader(f))

            # Suppression des en-têtes
            headers = hn[0]
            hn = hn[1:]

            # Séparation des publications `Ask HN` et `Show HN`
            ask_posts = []
            show_posts = []

            for post in hn:
                title = post[1]
                if title.lower().startswith("ask hn"):
                    ask_posts.append(post)
                elif title.lower().startswith("show hn"):
                    show_posts.append(post)
