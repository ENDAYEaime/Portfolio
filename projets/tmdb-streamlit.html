Analyse de Films TMDb avec Streamlit

Présentation du projet

Application web interactive développée avec Streamlit pour analyser et visualiser des données de films issues de la base de données TMDb (The Movie Database). Ce projet démontre l'utilisation de Streamlit pour créer des visualisations interactives et des tableaux de bord dynamiques.

  • Interface interactive : Tableau de bord dynamique avec filtres personnalisables
  • Visualisations : Graphiques interactifs avec Plotly et Seaborn
  • Analyse de données : Exploration approfondie des films et acteurs
  • Performance : Optimisation du chargement des données avec cache

Technologies utilisées

  • Streamlit : Framework pour l'interface utilisateur
  • Python : Langage principal
  • Pandas : Manipulation des données
  • Plotly & Seaborn : Visualisations interactives
  • Matplotlib : Visualisations statiques

Extrait du code


import streamlit as st
import pandas as pd
import plotly.express as px

st.set_page_config(page_title="Analyse de Films TMDb", layout="wide")

def load_data():
    movies_df = pd.read_csv('data/TMDb_Dataset.csv')
    actors_df = pd.read_csv('data/Birth_Actors.csv')
    return movies_df, actors_df

movies_df, actors_df = load_data()
st.title("Analyse de Films TMDb")

# Exemple de visualisation
fig = px.histogram(movies_df, x='vote_average', nbins=20, title="Distribution des notes des films")
st.plotly_chart(fig, use_container_width=True)
