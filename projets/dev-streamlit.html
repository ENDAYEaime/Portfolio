Développement d'Applications avec Streamlit

Présentation du projet

Projet d'apprentissage et de développement d'applications web interactives avec Streamlit. Ce projet couvre l'ensemble des fonctionnalités de Streamlit pour créer des applications de data science et de visualisation de données professionnelles.

  • Interface interactive : Création d'applications web dynamiques
  • Widgets avancés : Boutons, sliders, sélecteurs et plus
  • Visualisation de données : Intégration de graphiques et tableaux
  • Manipulation de données : Gestion efficace des DataFrames

Technologies utilisées

  • Streamlit : Framework principal
  • Python : Langage de programmation
  • Pandas : Manipulation des données
  • Matplotlib : Visualisations de base
  • Plotly : Visualisations interactives

Extrait du code


import streamlit as st
import pandas as pd
import matplotlib.pyplot as plt
import plotly.express as px

# Configuration de la page
st.set_page_config(page_title="Mon Application Streamlit")

# Création d'une sidebar
st.sidebar.title("Navigation")
option = st.sidebar.radio(
    "Choisissez une page:",
    ('Accueil', 'Exploration de données', 'Visualisation', 'À propos')
)

# Affichage conditionnel selon la sélection
if option == 'Accueil':
    st.header('Bienvenue sur la page d\'Accueil!')
    st.write('Choisissez une option dans la sidebar pour commencer.')
elif option == 'Exploration de données':
    st.header('Explorez vos données ici')
    # Widgets pour l'exploration de données
    uploaded_file = st.file_uploader("Téléchargez un fichier CSV", type="csv")
    if uploaded_file is not None:
        df = pd.read_csv(uploaded_file)
        st.dataframe(df)
