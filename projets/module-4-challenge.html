Analyse d'un Journal d'Appel

Présentation du projet

Ce projet consiste en l'analyse d'un journal d'appel pour extraire des informations pertinentes et générer des insights sur les habitudes d'appel. Les techniques d'analyse de données sont utilisées pour :

  • Analyser les tendances d'appel : identification des heures de pointe et des jours les plus actifs
  • Analyse statistique : création de tableau pour illustrer les tendances clés
  • Optimiser les ressources : recommandations basées sur les données pour améliorer l'efficacité

Technologies utilisées

  • Python : langage principal
  • Pandas : bibliothèque pour la manipulation de données
  • Jupyter Notebook : environnement de développement
