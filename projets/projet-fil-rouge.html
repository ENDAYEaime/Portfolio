Projet Fil Rouge – Analyse Morpho-Évolutive Humaine

Présentation du projet

Projet d'analyse et de visualisation de données paléoanthropologiques sur l'évolution humaine. Étude complète de la capacité crânienne et des traits morphologiques à travers le temps, avec analyses statistiques, visualisations avancées et synthèse scientifique.

  • Exploration complète : Nettoyage, structuration et analyse du dataset
  • Visualisations multiples : Boxplots, heatmaps, courbes temporelles
  • Corrélations & synthèse : Analyses statistiques, rapport scientifique
  • Gestion agile : Application de la méthode SCRUM

Technologies utilisées

  • Python : Langage principal
  • Pandas : Manipulation des données
  • Jupyter : Analyse interactive
  • Seaborn & Matplotlib : Visualisations
  • Agile : Gestion de projet SCRUM

Extrait du code


import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.read_csv('data/Evolution_DataSets.csv')

# Exemple : Visualisation de la capacité crânienne par espèce
plt.figure(figsize=(12, 6))
sns.boxplot(data=df, x='Genus_&_Specie', y='Cranial_Capacity')
plt.xticks(rotation=45)
plt.title('Distribution de la Capacité Crânienne par Espèce')
plt.tight_layout()
plt.show()
