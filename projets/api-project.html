Projet API

Présentation du projet

Ce projet explore l'utilisation des API pour récupérer et manipuler des données en temps réel. Il inclut des exemples d'intégration d'API tierces et de création d'une API simple avec Flask.

  • Intégration d'API tierces : récupération de données en temps réel
  • Création d'une API : développement avec Flask
  • Manipulation des données : traitement et visualisation des données récupérées

Technologies utilisées

  • Python : langage principal
  • Flask : framework pour créer des API
  • Requests : bibliothèque pour les requêtes HTTP
  • Jupyter Notebook : environnement de développement
