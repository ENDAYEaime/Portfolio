Analyse de Données Cinématographiques

Présentation du projet

Projet réalisé dans le cadre du Module 3 : Challenge 2. L'objectif était d'explorer, nettoyer et analyser une base de données cinématographique (type Sakila) à l'aide de Python, SQL et Jupyter Notebook. Le projet comprend :

  • Chargement et structuration des données : import de fichiers CSV dans une base SQLite, création des tables et jointures
  • Nettoyage des données : gestion des valeurs manquantes, suppression des doublons, vérification des types
  • Analyse exploratoire : requêtes SQL avancées pour extraire des statistiques sur les films, clients, locations, revenus, etc.
  • Visualisation : utilisation de pandas et matplotlib/seaborn pour représenter les résultats

Technologies utilisées

  • Python : manipulation et analyse de données
  • Pandas : traitement des DataFrames
  • SQLAlchemy : gestion de la base SQLite
  • Jupyter Notebook : environnement interactif
  • SQL : requêtes analytiques

Extrait du code


# Chargement des CSV dans SQLite
import pandas as pd
from sqlalchemy import create_engine

engine = create_engine('sqlite:///data/database.db')
csv_files = {
    'actor.csv': 'actor',
    'film.csv': 'film',
    # ... autres fichiers ...
}
for csv_file, table_name in csv_files.items():
    df = pd.read_csv(f'data/{csv_file}')
    df.to_sql(table_name, con=engine, if_exists='replace', index=False)

# Exemple de requête SQL pour les films les plus populaires par genre
query = '''
SELECT c.name AS category, f.title, COUNT(i.inventory_id) AS inventory_count
FROM film f
JOIN film_category fc ON f.film_id = fc.film_id
JOIN category c ON fc.category_id = c.category_id
JOIN inventory i ON f.film_id = i.film_id
GROUP BY c.name, f.title
ORDER BY inventory_count DESC
'''
pd.read_sql(query, engine)
