Guide d'Apprentissage Jupyter Notebook

Présentation du projet

Apprendre et maîtriser Jupyter Notebook, un outil essentiel en data science. Ce tutoriel couvre :

  • Exécution de code : manipulation des cellules et exécution du code Python
  • Raccourcis clavier : optimisation du flux de travail avec les commandes essentielles
  • Gestion d'état : compréhension de l'état du kernel et des variables
  • Markdown : création de documentation interactive avec du texte formaté

Environnement technique

  • Jupyter Notebook : environnement interactif
  • Python 3 : langage de programmation
  • Markdown : documentation

Extrait du code


def welcome(a_string):
    welcome_msg = 'Welcome to ' + a_string + '!'
    return welcome_msg

# Exemples d'utilisation
dq = 'Dataquest'
jn = 'Jupyter Notebook'
py = 'Python'

# Démonstration des résultats
print(welcome(dq))    # Welcome to Dataquest!
print(welcome(jn))    # Welcome to Jupyter Notebook!
print(welcome(py))    # Welcome to Python!

# Exemple de manipulation de données
opened_file = open('AppleStore.csv')
from csv import reader
read_file = reader(opened_file)
apps_data = list(read_file)

# Affichage des premières lignes
header = apps_data[0]
first_row = apps_data[1]
print("\nColonnes :", header)
print("Premier app :", first_row)
